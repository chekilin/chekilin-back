# 実行
**コンテナを立ち上げたいとき**
```md
make　up
```

**コンテナを終了したいとき**
```md
make　down
```

**キャッシュを処分して、もう一度コンテナを作り直したいとき**
```md
make restart
```

**ログを表示したいとき**
```md
make logs
```

**ヘルプ**
```md
make help
```
